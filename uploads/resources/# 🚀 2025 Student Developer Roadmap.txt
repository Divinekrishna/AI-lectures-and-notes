### **Day-by-Day Plan**

#### **Day 1: Jan 13 (Today) - The Setup**

* **DSA (45m):** Topic: **Arrays**.
* *Problem:* "Move Zeroes" (LeetCode 283). Focus on moving elements without making a copy.


* **ML/PyTorch:**
* **Goal:** Understand **Tensors**. (They are just Numpy arrays that run on GPU).
* **Task:** Install PyTorch. Create random tensors, reshape them (`.view()`), and do basic math (add, multiply).
* **Resource:** `torch.tensor`, `torch.randn`.


* **SWOC:** Join the Discord servers of 2-3 organizations. Introduce yourself.

#### **Day 2: Jan 14 - The Magic of Autograd**

* **DSA (45m):** Topic: **Strings**.
* *Problem:* "Valid Anagram" (LeetCode 242). Use a Hash Map (dictionary).


* **ML/PyTorch:**
* **Goal:** Understand **Gradient Descent**.
* **Concept:** How `loss.backward()` calculates gradients automatically. You don't need to do calculus; PyTorch does it for you.
* **Task:** Implement "Linear Regression" from scratch using PyTorch (predicting `y = 2x + 1`).



#### **Day 3: Jan 15 - The Training Loop (Crucial)**

* **DSA (45m):** Topic: **Two Pointers**.
* *Problem:* "Squares of a Sorted Array" (LeetCode 977).


* **ML/PyTorch:**
* **Goal:** Memorize the 5 Steps.
* **Code:**
1. `optimizer.zero_grad()`
2. `output = model(input)` (Forward pass)
3. `loss = criterion(output, target)` (Calculate error)
4. `loss.backward()` (Calculate gradients)
5. `optimizer.step()` (Update weights)


* **Task:** Train a tiny network on dummy data.



#### **Day 4: Jan 16 - Computer Vision Basics (CNNs)**

* **DSA (45m):** Topic: **Sliding Window**.
* *Problem:* "Max Sum Subarray of size K" (The one we discussed).


* **ML/PyTorch:**
* **Goal:** Move beyond numbers to **Images**.
* **Task:** Load the **MNIST Dataset** (Handwritten digits) using `torchvision`.
* **Project:** Build a simple classifier that predicts if an image is a '7' or a '1'.



#### **Day 5: Jan 17 - SWOC Action Day**

* **DSA (45m):** Topic: **Binary Search**.
* *Problem:* "Binary Search" (LeetCode 704) - Standard implementation.


* **SWOC Focus:**
* **Goal:** Make a Pull Request (PR).
* **Strategy:** Go to a Python-based repo. Look for a folder called `scripts` or `utils`.
* **Action:** Write a script (e.g., "Image Resizer using Pillow" or "CSV to JSON converter") and submit it.
* *Note:* Even if it's simple, getting a PR merged boosts confidence.



#### **Day 6: Jan 18 - Finishing Touches**
